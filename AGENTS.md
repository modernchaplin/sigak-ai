# SIGAK.AI Project Rules

- Keep the product centered on the five axes: 시간, 사람, 지도, 기록, 시각.
- User-facing site name must be `SIGAK.AI`.
- Do not add login, DB, search, community, admin, crawling, AI generation, or external API features unless explicitly requested.
- Content should live in data files, not scattered through components.
- Topic detail pages should use the shared data-driven template: summary, tags, sections, optional timeline items, and related content.
- Preserve the black background, white text, yellow accent, calm archive-like tone.
- Prefer small, maintainable improvements over large rewrites.
- Verify with `npm run lint` and `npm run build` when possible before calling work complete.
